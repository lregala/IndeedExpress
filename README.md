# IndeedExpress
A python job posting scraper for <a href="https://ca.indeed.com/">Indeed<a>. 
Outputs an xlsx file containing the scraped postings.

Packages required:
<ul>
  <li>
    Beautiful Soup: 
    <code>
      $ pip install beautifulsoup4
    </code>
  </li>
  <li>
    Requests: 
    <code>
      $ pip install requests
    </code>
  </li>
  <li>
    lxml: 
    <code>
      $ pip install lxml
    </code>
  </li>
   <li>
    openpyxl: 
    <code>
      $ pip install openpyxl
    </code>
  </li>
   <li>
    pandas: 
    <code>
      $ pip install pandas
    </code>
  </li>
 </ul>
